# Fractal Explorer — Browser Edition

An interactive Mandelbrot/Julia set explorer that runs entirely in your browser. No installation, no dependencies — just open the HTML file.

## Features

- Mandelbrot & Julia set modes
- Orbit tracer with animated path rendering
- 8 color palettes: Red CRT, Amber CRT, Hybrid, Fire, Ice, Ultra, Psychedelic, Grayscale
- 6 presets: Seahorse Valley, Elephant Valley, Double Spiral, Tendrils, Mini Brot, Lightning
- Drag to draw zoom box, Shift+drag to pan, scroll to zoom
- Animated orbit tracer with speed control
- Julia set preview panel on hover
- Save PNG with orbit overlay composited
- Full CRT phosphor aesthetic

## Usage

Open `index.html` in any modern browser. No server required.

## Controls

| Input | Action |
|---|---|
| Drag | Draw zoom box |
| Shift + Drag | Pan |
| Scroll | Zoom in/out |
| J | Toggle Julia mode |
| O | Toggle orbit tracer |
| C | Clear orbit |
| S | Save PNG |
| R | Re-render |
| ESC | Reset view |
